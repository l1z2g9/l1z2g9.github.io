  • Download ‘setup.exe’ from Cygwin website
  • Download hadoop 0.20.2 and hbase 0.20.6 from apache website

  • Add sshd service
  • Generate ssh key
  • Edit file as below
    • core-site.xml
    • hdfs-site.xml
    • mapred-site.xml
    • hadoop-env.sh

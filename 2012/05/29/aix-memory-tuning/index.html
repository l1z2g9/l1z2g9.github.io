aix memory tuning

真实内存数量

file pages, minperm, maxperm, numperm关于文件页面占用内存，跟non-computational内存有关(topas)

client pages,maxclient,numclient关于nfs被client连接使用内存有关

数据库服务器由于是计算密集型，所以minperm, maxperm percentage可以设置小一些，而文件服务器可以设置大一些的。

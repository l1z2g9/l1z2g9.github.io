在MSSQL Server 2008中创建可以访问外部资源的DLL

编写clr函数注意：

  1. 静态类型static修饰的函数
  2. 如果函数内部不执行sql查询，应定义DataAccessKind.None
  3. 最好定义IsDeterministic = true。但如果是DataAccessKind.READ，定义确定性函数也是没有用的。
